# Caf-Hop
CaféHop is a web app that helps students discover cafés based on budget, rating, and ambience. It uses a public API with JavaScript fetch and allows users to search, filter, and sort cafés using array methods, with a responsive and user-friendly interface.

## 📌 Project Description

CaféHop is a web-based café discovery application designed for students to find cafés based on budget, rating, and ambience. It helps users explore suitable places for studying, hanging out, or casual meetups.

## 🌐 API Used

  Overpass API (OpenStreetMap) will be used to fetch real-time café data such as location, ratings, and details.

## 🚀 Planned Features

* Search cafés by name or location
* Filter cafés based on budget and category
* Sort cafés by rating or popularity
* Responsive user interface

## 🛠️ Technologies

* HTML
* CSS / Tailwind CSS
* JavaScript (Fetch API)

## ⚙️ Setup Instructions

1. Clone the repository
2. Open the project folder
3. Run index.html in browser
